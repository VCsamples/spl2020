There are 5 systems implemented for comparison:

Reference Systems:
(a) MT-WORLD:
This is a generator-vocoder pipeline system. The generator with multi-task learning is followed by a WORLD parametric vocoder.
(b) MT-WaveRNN:
This is a generator-vocoder pipeline system. The generator with multi-task learning is followed by a WaveRNN neural vocoder.
(c) STWaveRNN:
This is an integrated architecture that takes linguistic features as input and generates speech waveform as output. As the whole network is shared for both languages, it is called Single-Task WaveRNN or STWaveRNN.
Proposed Systems:
(d) MTWaveRNN:
This is an integrated architecture, that takes linguistic features as input andgenerates speech waveform as output directly. Each language-dependent head in the output section has two dense layers.
(e) MTWaveRNN+:
This is an integrated architecture, that takes linguistic features as input andgenerates speech waveform as output directly. Each language-dependent head in the output section has three dense layers in each language-dependent. We use a ‘+’ sign to indicate there is an extra dense layer.
